Claim your territory:
```
  Add a comment before your code in the format of:
    'NAME week_#'
```
Line Spacing:
```
  Two tabs per indentation (2 spaces is what this works out to be in tic80)
  
  Align start/end for readability
```

Variable Case Types:
```
  general (functions/variables) - 'snake_case'
  classes - 'PascalCase'
```

Data Type Prefixes:
```
  nil - 'n_'
  boolean - 'b_'
  number - 'i_'
  string - 's_'
  function - NONE
  userdata - 'u_'
  thread - 'T_'
  table - 'a_'
```

Useful Tokens (from ref. manual):
```
Reserved Names:
  and       break     do        else      elseif    end
  false     for       function  goto      if        in
  local     nil       not       or        repeat    return
  then      true      until     while

Reserved Tokens:
   +     -     *     /     %     ^     #
   &     ~     |     <<    >>    //
   ==    ~=    <=    >=    <     >     =
   (     )     {     }     [     ]     ::
   ;     :     ,     .     ..    ...
     
Arithmatic Operators:
  +: addition
  -: subtraction
  *: multiplication
  /: float division
  //: floor division
  %: modulo
  ^: exponentiation
  -: unary minus

Bitwise Operators:
  &: bitwise AND
  |: bitwise OR
  ~: bitwise exclusive OR
  >>: right shift
  <<: left shift
  ~: unary bitwise NOT
  
Relational Operators:
  ==: equality
  ~=: inequality
  <: less than
  >: greater than
  <=: less or equal
  >=: greater or equal
  
```
