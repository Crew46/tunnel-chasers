Useful Tokens (from ref. manual):
```
Reserved Names:
  and       break     do        else      elseif    end
  false     for       function  goto      if        in
  local     nil       not       or        repeat    return
  then      true      until     while

Reserved Tokens:
   +     -     *     /     %     ^     #
   &     ~     |     <<    >>    //
   ==    ~=    <=    >=    <     >     =
   (     )     {     }     [     ]     ::
   ;     :     ,     .     ..    ...

Arithmatic Operators:
  +: addition
  -: subtraction
  *: multiplication
  /: float division
  //: floor division
  %: modulo
  ^: exponentiation
  -: unary minus

Bitwise Operators:
  &: bitwise AND
  |: bitwise OR
  ~: bitwise exclusive OR
  >>: right shift
  <<: left shift
  ~: unary bitwise NOT

Relational Operators:
  ==: equality
  ~=: inequality
  <: less than
  >: greater than
  <=: less or equal
  >=: greater or equal

```
